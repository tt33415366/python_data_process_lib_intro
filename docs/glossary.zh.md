# 术语词汇表

本文档为“Python 科学计算演进”文档库中使用的常见术语、概念和技术提供详细定义。

---

### A

**API (Application Programming Interface / 应用程序编程接口)**
: API 是一个明确定义的契约，规定了软件组件之间应如何交互。它暴露了一组开发者可以使用的函数、类、方法和常量，同时将内部的复杂性抽象出来。例如，当您使用 Pandas 库的 `pd.DataFrame()` 时，您正在使用其 API 来创建一个 DataFrame 对象，而无需了解其内部实现的复杂细节。

**Array (数组)**
: 一种数据结构，它将一组元素（通常是相同类型）存储在连续的内存块中，可通过一个索引或一组索引元组来访问。在科学计算中，多维数组（如 NumPy 的 `ndarray`）是数值运算的基础，因为它们支持高效的向量化计算。

**Attention Mechanism (注意力机制)**
: 现代神经网络中的一项关键创新，尤其是在像 Transformer 这样的自然语言处理模型中。它允许模型在处理特定元素时，权衡输入序列中不同部分的重要性。例如，在翻译一个句子时，注意力机制帮助模型为它生成的每个目标词汇专注于相关的源语言词汇。

**Asynchronous Programming (异步编程)**
: 一种允许任务独立于主程序流程运行的编程范式，从而实现并发。在 Python 中使用 `async` 和 `await` 关键字，程序可以执行耗时较长的操作（如 I/O 请求）而不会阻塞其他任务，从而提高整体效率和响应能力。

---

### B

**Backend (后端)**
: 高级库用来执行其操作的底层计算引擎。后端可以是软件（例如，一个使用 Matplotlib 绘制图形的绘图库），也可以是硬件（例如，PyTorch 通过 CUDA 使用 GPU 后端进行加速的张量计算）。这种抽象允许用户编写一次代码，然后在不同的平台上运行。

**Backpropagation (反向传播)**
: 用于训练人工神经网络的核心算法。它的工作原理是计算损失函数相对于网络权重的梯度（或导数）。然后，优化算法（如梯度下降）使用这个梯度来更新权重，以最小化损失。它本质上是将误差信号从输出层向后传播到输入层。

---

### C

**Concurrency (并发)**
: 系统在重叠的时间段内执行多个任务或进程的能力，但不一定同时执行。这通常是通过在单个 CPU 核心上进行时间分片来实现的。并发对于构建能够处理多个用户请求或后台进程的响应式应用程序至关重要。另见: **Parallelism (并行)**。

**Convolution (卷积)**
: 对两个函数进行的数学运算，产生第三个函数，表示其中一个函数的形状如何被另一个函数修改。在卷积神经网络 (CNN) 中，卷积涉及在一个输入图像上滑动一个小滤波器（或称为核），以生成一个特征图。此操作在检测边缘、纹理和形状等模式方面非常有效。

---

### D

**DataFrame**
: 一种二维的、大小可变的、并且可能包含异构数据的表格型数据结构，具有带标签的轴（行和列）。它是 Pandas 库中的核心数据结构，其灵感来源于 R 语言的 data.frame。它非常适合处理和分析结构化数据，类似于电子表格或 SQL 表。

**Deep Learning (深度学习)**
: 机器学习的一个子领域，基于具有许多层次（因此称为“深”）的人工神经网络。通过堆叠层次，这些模型可以学习数据的分层表示，自动发现复杂的模式，而无需手动进行特征工程。它推动了计算机视觉、自然语言处理等领域的重大突破。

---

### F

**Framework (框架)**
: 一套全面的、集成的工具、库和约定，为应用程序开发提供了结构化的基础。像 PyTorch 和 TensorFlow 这样的框架为构建、训练和部署机器学习模型提供了完整的生态系统。它们通常遵循“控制反转”原则，即框架的代码调用开发者的自定义代码，而不是相反。

**Library (库)**
: 可重用代码（函数、类）的集合，提供特定的功能。与框架不同，库通常用于执行一组相关的任务，并且由开发者的代码控制，在需要时调用库的函数。例如，NumPy 是一个用于数值计算的库，而 Requests 是一个用于发出 HTTP 请求的库。

---

### G

**Gradient Descent (梯度下降)**
: 一种用于寻找函数最小值的迭代优化算法。在机器学习中，它通过调整模型参数来最小化模型的损失函数。在每一步中，它计算损失函数的梯度，并朝着负梯度的方向迈出一步，这对应于最陡峭的下降方向。

---

### L

**Lazy Evaluation (惰性求值)**
: 一种评估策略，其中表达式在绑定到变量时不会立即计算，而是将其执行推迟到明确需要其结果时。像 Dask 和 Spark 这样的库使用这种策略来构建一个操作的计算图，然后可以在执行前对整个图进行优化，从而带来显著的性能提升，尤其是在处理大型数据集时。

---

### M

**Machine Learning (ML) (机器学习)**
: 人工智能的一个领域，它赋予计算机从数据中学习的能力，而无需进行显式编程。机器学习算法基于样本数据（称为“训练数据”）构建一个数学模型，以便对新的、未见过的数据进行预测或决策。

**Model (模型)**- 
: 在机器学习中，模型是训练过程的输出。它是一个文件，包含一个算法和一组学习到的参数（权重和偏置），可用于对新数据进行预测。例如，在用猫的图片训练一个神经网络后，得到的模型可以预测一张新图片是否包含猫。

---

### N

**Neural Network (神经网络)**
: 一种受生物神经网络结构和功能启发的计算模型。它由称为“神经元”的互连节点组成，这些节点组织在层次中。每个连接都有一个在训练过程中调整的“权重”。数据从输入层流经一个或多个隐藏层，最终到达输出层，产生最终的预测。

**Natural Language Processing (NLP) (自然语言处理)**
: 人工智能的一个领域，专注于使计算机能够理解、解释和生成人类语言。NLP 任务包括翻译、情感分析、文本摘要和问答。现代 NLP 在很大程度上依赖于像 Transformer 这样的深度学习模型。

---

### P

**Parallelism (并行)**
: 系统同时执行多个任务或进程的能力，通常通过将它们分布在多个 CPU 核心甚至多台机器上实现。这对于高性能计算和处理“大数据”至关重要。另见: **Concurrency (并发)**。

**Plotting (绘图)**
: 创建数据图形表示的行为。像 Matplotlib 和 Seaborn 这样的可视化库提供了广泛的绘图函数，用于创建图表、图形和图像（例如，折线图、条形图、散点图），以帮助理解数据和传达发现。

**Pooling (池化)**
: 在卷积神经网络 (CNN) 中常用的一种操作，用于减少特征图的空间维度（宽度和高度）。它通过聚合局部区域的特征来实现，例如，通过取最大值（最大池化）或平均值（平均池化）。池化有助于使学习到的表示对小的平移更具鲁棒性，并减少计算复杂性。

---

### S

**Series (序列)**
: 一种能够容纳任何数据类型（整数、字符串、浮点数、Python 对象等）的一维带标签数组。它是 Pandas 库中用于存储单列或单行数据的主要数据结构。一个 DataFrame 本质上是一个 Series 的集合。

---

### T

**Tensor (张量)**
: 深度学习中使用的主要数据结构。它是向量和矩阵到任意维度的推广。张量由其阶（rank，即维度数）、形状（shape，即每个维度的大小）和数据类型来标识。
- **Rank 0**: 标量（一个单独的数字）。
- **Rank 1**: 向量（一个一维数组）。
- **Rank 2**: 矩阵（一个二维数组）。
- **Rank 3 及更高**:更高维的张量。

---

### V

**Vectorization (向量化)**
: 用数组或向量上的操作替换显式循环的过程。向量化操作速度更快，因为它们是用低级语言（如 C 或 Fortran）实现的，并且可以利用现代 CPU 的能力（如 SIMD 指令）。NumPy 是一个在 Python 中启用和鼓励向量化的典型库。
